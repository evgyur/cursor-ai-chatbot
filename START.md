# Cursor AI Chatbot — Промпт для запуска

```
Примени скилл cursor-ai-chatbot для создания AI-чатбота на сайте.

Проведи со мной диалог:

1. "Введите ваш MiniMax API Key (получить на https://platform.minimax.io):"
   - Ключ будет зашит в server.js

2. "Как зовут консультанта?"

3. "Расскажите о ваших услугах и ценах:"

После ответов создай:
1. server.js — с зашитым API ключом
2. knowledge.json — услуги и цены
3. chat-widget.html — с именем консультанта
4. package.json — с зависимостями (express, cors)

Покажи как запустить.
```
