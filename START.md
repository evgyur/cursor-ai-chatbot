# Cursor AI Chatbot — Промпт для запуска

Скопируй этот промпт и отправь в Cursor:

---

```
Примени скилл cursor-ai-chatbot для создания AI-чатбота на сайте.

Проведи со мной диалог и задай следующие вопросы (по очереди):

1. "Введите ваш MiniMax API Key (получить на https://platform.minimax.io):"

2. "Введите Group ID из панели MiniMax:"

3. "Как зовут консультанта? (по умолчанию: Консультант)"

4. "Расскажите о ваших услугах и ценах — что вы предлагаете и по каким ценам? (или дайте ссылку на сайт с ценами):"

После получения ВСЕХ ответов:

1. Создай файл server.js:
   - Подставь API Key и Group ID в код
   - Добавь system prompt с правилами (не галлюцинировать, отвечать по базе)

2. Создай файл knowledge.json:
   - Запиши услуги и цены в JSON формате

3. Обнови chat-widget.html:
   - Подставь имя консультанта
   - Настрой приветственное сообщение

4. Создай package.json с зависимостями (express, cors)

5. Покажи инструкцию как запустить и проверить.
```
