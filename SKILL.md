# Cursor AI Chatbot Skill

Добавляет AI-чатбот на сайт. Спрашивает всё необходимое интерактивно.

## Активация

Скачай и открой в Cursor:
```bash
git clone https://github.com/evgyur/cursor-ai-chatbot.git
cd cursor-ai-chatbot
```

## Что спросит Cursor при применении

После активации скилла, Cursor задаст тебе вопросы:

### 1. MiniMax API ключ
```
Введите ваш MiniMax API Key:
```
Нужен для работы AI. Получить на https://platform.minimax.io

### 2. Group ID
```
Введите Group ID:
```
Берется из панели MiniMax

### 3. Имя консультанта
```
Как зовут консультанта? (по умолчанию: Консультант)
```

### 4. Данные об услугах
```
Расскажите о ваших услугах и ценах:
```
Можно просто написать текстом, например:
"Услуга 1: консультация 5000 руб, Услуга 2: аудит сайта 15000 руб"

ИЛИ дать ссылку на сайт откуда собрать информацию.

## Что получишь

После ответов на вопросы Cursor сгенерирует:
- ✅ Настроенный `chat-widget.html` с твоим именем консультанта
- ✅ Готовый `server.js` с твоим API ключом (зашит)
- ✅ Базу знаний с твоими услугами

Просто вставь виджет на сайт — всё работает.

## Без API ключа

Если ключа нет — можно использовать демо-режим с преднастроенным ключом (лимиты).

## Файлы после генерации

```
cursor-ai-chatbot/
├── chat-widget.html   # Виджет (готов к вставке на сайт)
├── server.js          # Backend сервер
└── knowledge.json     # Твои услуги и цены
```

## Запуск

```bash
npm install
npm start
```

Чат появится в правом нижнем углу.
