# Cursor AI Chatbot Skill

Добавляет AI-чатбот на сайт. Работает через диалог с пользователем.

## Как работает

Когда ты применяешь этот скилл в Cursor, он задаст вопросы и сгенерирует готовый код.

---

## Промпт для Cursor

```
Примени скилл cursor-ai-chatbot. Следуй инструкциям:

1. Спроси: "Введите ваш MiniMax API Key (получить на https://platform.minimax.io):"
   - Ключ будет зашит в server.js

2. Спроси: "Как зовут консультанта? (по умолчанию: Консультант)"

3. Спроси: "Расскажите о ваших услугах и ценах:"

После получения ответов:
- Создай server.js с подставленным API ключом
- Создай knowledge.json с данными об услугах
- Обнови chat-widget.html с именем консультанта
```

---

## Что получишь

- `server.js` — backend с зашитым API ключом
- `knowledge.json` — твои услуги и цены
- `chat-widget.html` — виджет для сайта

Вставь виджет на сайт, запусти сервер — готово!
