# Cursor AI Chatbot Skill

Добавляет AI-чатбот на сайт. Работает через диалог с пользователем.

## Как работает

Когда ты применяешь этот скилл в Cursor, он задаст тебе вопросы и сгенерирует готовый код.

---

## Промпт для Cursor

Скопируй этот промпт и отправь Cursor:

```
Примени скилл cursor-ai-chatbot. Следуй инструкциям:

1. Спроси: "Введите ваш MiniMax API Key (получить на https://platform.minimax.io):"
2. Спроси: "Введите Group ID из панели MiniMax:"
3. Спроси: "Как зовут консультанта? (по умолчанию: Консультант)"
4. Спроси: "Расскажите о ваших услугах и ценах (или дайте ссылку на сайт):"

После получения ответов:
- Создай server.js с подставленным API ключом и Group ID
- Создай knowledge.json с данными об услугах
- Обнови chat-widget.js с именем консультанта
```

---

## Что получишь

После диалога Cursor сгенерирует:

- `server.js` — backend с твоим API ключом
- `knowledge.json` — твои услуги и цены
- `chat-widget.html` — виджет для сайта

Просто вставь на сайт и запусти `npm start`.

---

## Альтернатива: без ключа

Если нет MiniMax — используй демо-режим (ограничения):

```javascript
// В chat-widget.js
const DEMO_MODE = true;
```
